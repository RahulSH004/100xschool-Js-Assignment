# JavaScript Assignments

This repository contains JavaScript assignments across different topics and difficulty levels.

## Assignment Categories

### 1. [js-basics](js-basics/README.md)
Basic JavaScript concepts and fundamentals.

### 2. [js-built-ins](js-built-ins/README.md)
JavaScript built-in methods and functions.

### 3. [01-js](01-js/README.md)
Core JavaScript assignments (easy, medium, hard).

### 4. [01-async-js](01-async-js/README.md)
Asynchronous JavaScript assignments including:
- **Easy:** Counter, Read/Write files
- **Medium:** File Cleaner, Clock
- **Hard:** Promisify setTimeout, Sleep, Promise all, Promise Chain

---

## Development Setup

### Option 1: Run Locally (Requires Node.js)
1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Clone or download this repository.
3. Navigate to the respective assignment folder and execute the script using Node.js.

### Option 2: Run on Replit (No Local Setup Required)
1. Copy the assignment code to [Replit](https://replit.com/).
2. Execute the script in Replit's environment.

---

## Running Tests

### For Hard-Level Assignments
1. Navigate to the assignment folder (e.g., `/01-async-js`).
2. Install dependencies using:
   ```sh
   npm install
   ```
3. Run specific tests with:
   ```sh
   npm run <test-name>
   ```
   Example:
   ```sh
   npm run 1-promisify-setTimeout
   ```
4. To execute all tests at once, run:
   ```sh
   npm run all
   ```

### For Easy & Medium Assignments
- There are no automated tests.
- Manually run scripts using Node.js:
  ```sh
  node easy/1-counter.js
  ```
  (Replace with the relevant file name.)
